What is Redux?

↠Redux was created by Dan Abramov and Andrew Clark and was inspired by the Flux architecture from Facebook.

↠Redux is a popular JavaScript library for managing and centralizing application state.

↠It is most commonly used with libraries such as React or Angular but can be used with any other JavaScript framework or library.

What is Redux Thunk used for?

↠ Redux Thunk is a middleware for Redux that allows you to write action that return a function instead of an action.

↠you use Redux Thunk, an action returns a function instead of an action object.

↠ This returned function can perform asynchronous tasks and can also dispatch multiple actions.

What is the second argument that can optionally be passed tosetState and what is its purpose?

↠In React, the second argument that can be optionally passed to setState is a callback function.

↠ Its purpose is to execute code immediately after the state has been updated and the component has been re-rendered

↠ you using the callback function, you can access the updated state and perform actions accordingly
